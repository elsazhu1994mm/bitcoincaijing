# 如何把欧易Web3钱包切换到TRON主网？

你是否在用欧易Web3钱包，收USDT的时候发现选错了网络，或者明明钱包里面有USDT却转不了账？很可能是你还没有切换到TRON网络，或者缺少能量。本篇文章就来教你两个关键操作。

**1、如何把欧易Web3钱包切换到TRON主网接收USDT-TRC20**

**2、如何给钱包充值TRX能量，顺利转账。**

# **一、切换主网接收TRC20**

首先我们打开欧易APP，在上方交易所这里切换到我们的Web3钱包。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/03102ad7d7_1749533853-5.jpg) 

点击右上角地区图标。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1cbd0d597f_1749537759-2.jpg) 

选择下方的“TRON”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ea4aa4de29_1749537844-3.jpg) 

成功之后我们的钱包地址就会变成以T开头，表示是TRON主网地址。然后我们点击这里的【接收】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a54b5b4707_1749534669-16.jpg) 

选择USDT。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/41d30e954f_1749537956-4.jpg) 

这个时候我们钱包地址就显示出来了，复制这个地址提供给他人转账就可以了。需要提示一下，只有在TRON网络下的USDT才是TRC20格式。其他的比如说以太坊网络是ERC20，我们千万不能混用。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7df6b37583_1749538089-5.jpg) 

# **二、充值能量**

这个时候需要我们先回到交易所。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a0d7b9431b_1749536501-29.jpg) 

来到下方的交易，点击两次，选择【现货】交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9cedf9d435_1749538328-6.jpg) 

左上方这里我们就可以搜索TRX，选择“TRX/USDT”这个交易对。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/66ca4eb6df_1749538444-9.jpg) 

在这里我们就可以购买一些TRX。在买入的界面，点击【买入TRX】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/35b350bc46_1749538538-10.jpg) 

这时候我们的资产里面就有TRX这个币了，我们就可以把它发送到我们的钱包，我们可以点击TRX。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/51a1e46e1e_1749538615-11.jpg) 

选择【提币】，选择“提取数字货币”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1bfab8d0bc_1749538694-14.jpg) 

选择“其他交易所或钱包”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4c7f6bb575_1749538735-15.jpg) 

确保我们是通过TRON这条链，来提取这个TRX到我们的钱包的。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5b2ea96338_1749538796-16.jpg) 

粘贴上我们刚刚复制的地址，输入我们想要提的TRX的数量，这里最小提币额是20.8个。可以看到这里有一个网络费用是0.8TRX，无论我们提多少这个网络费用它是不会变的，都是收取这么多。点击【提交】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ef47e092be_1749538945-17.jpg) 

确认一下我们的地址和网络没有问题。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/cb74153653_1749539001-18.jpg) 

这时候我们的能量就正在发送中了，需要我们耐心等待一下。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0650584bd2_1749539056-19.jpg) 

回到我们的钱包，可以看到我们充值的能量TRX已经到账了。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a54b1c417c_1749539113-20.jpg) 

还有一种方法是，你可以直接在这里点击【接收】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/fed07709fd_1749539166-21.jpg) 

选择“从交易所账户快捷提币”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/af638186d0_1749539210-22.jpg) 

选择“TRX”，提币网络自然就是TRON。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/02aaceb705_1749539302-25.jpg) 

你可以选择关联钱包至交易所账户，确认一下。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8e981ff672_1749539407-28.jpg) 

这时候它就给我们自动填写上了充值的地址，我们直接输入想要提币的金额，再点击【提交】也是一样的，能够充值我们的能量。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/209c5f20c2_1749539494-29.jpg)
