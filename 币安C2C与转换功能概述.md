# 币安C2C和转换功能介绍

今天我们将介绍C2C和转换功能，以及如何在钱包之间转移币种。在此之前，我将告诉你如何以20%的手续费折扣进行交易。方法非常简单，只需要点击下方展示的20%手续费折扣链接进行注册，即可节省手续费进行交易。

# **C2C使用方法**

首先我们来到币安首页来了解基本功能。点击左上角的【一键买币】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/fb9af4208c_1720680846-29.jpg) 

进入C2C交易区。“C2C”指买家和卖家直接交易加密货币的方法。在左上方您会看到根据交易量划分的不同区域。“快捷区”支持快速交易。“自选区”是您选择和收集偏好加密货币的地方。“大宗区”适用于大规模交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/88c607f4be_1720681022-30.jpg) 

如需小额兑换，请使用快捷区。如需交换偏好货币，请使用自选区。如需大量购买，请使用大宗区。

选择C2C时，先确定买入或卖出操作。我们从买入选项开始。在“我要支付”中将基础货币设置为人民币，在“我将收到”中设置USDT。“USDT”是稳定币，意味着其价值相对稳定。USDT的价值为1美元=1USDT，点击【0手续费购买】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3b89fc57a0_1720681368-31.jpg) 

选择您的付款方式并进行付款。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/56881a7bdc_1720681507-32.jpg) 

选择完成后，币安将随机为您连接卖家，并在右侧显示聊天窗口。您可使用该窗口与卖家私密对话。在聊天窗口左侧能看到提示：请按以下交易方式付款给对方。这里显示付款方式、卖家姓名和账户信息及收款二维码。请使用提供的信息完成付款。付款后，点击【我已完成付款，通知对方】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3db1a6ff79_1720682034-33.jpg) 

完成这些步骤后，卖家发送加密货币后，人民币将成功转换为USDT。查看USDT是否到账的方法是：返回币安主页，点击右上角钱包图标，选择“现货账户”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a275b31810_1720682402-34-1024x475.jpg) 

向下滚动页面，您能看到成功存入的USDT。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d3bd4a4935_1720682476-35.jpg) 

# **币安闪兑介绍**

接下来介绍币安闪兑功能。您可通过币安闪兑直接进行币种兑换，无需现货交易。转换过程不收取交易费用，但请注意存在点差。点差指购买时币种价格高于市场价格，而卖出时价格低于市场价格的情况。金额增加时，点差也会增大。因此，少量币种转换时建议使用币安闪兑功能。下面介绍使用方法。

首先登录币安账户。在【交易】部分点击【币安闪兑】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/a487f9b2c8_1720683029-36.jpg) 

选择按市价或限价交易。在右侧选择持有币种的钱包。选择当前持有币种，再选择目标兑换币种。点击预览兑换结果进行兑换即可。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/f8f52abd14_1720683216-37.jpg) 

# **数字货币钱包转移方法**

下面介绍如何在钱包之间转移资金。点击右上角钱包图标，选择“现货账户”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/08b03f3ff9_1720683337-38.jpg) 

在右侧找到并点击【划转】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/14c2bfe0d1_1720683391-39.jpg) 

选择当前持有币种的钱包和目标转移钱包。例如，从现货账户转移100USDT到资金钱包（U本位合约），则在“从”选择“现货账户”，在“至”选择“U本位合约”，搜索并选择USDT作为币种。如需转移全部资金，点击“全部”，确认后转移完成。您可查看现货账户中的100USDT已转移到资金钱包（U本位合约）。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0657defaa9_1720683681-40.jpg)
