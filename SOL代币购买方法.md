# 如何购买SOL代币？

本篇文章将为你介绍如何使用币安交易所购买SOL代币的详细教程。

# **第一步：币安账户注册**

1. 点击下方币安交易平台注册链接，进入注册页面。

**<https://www.binance.com/zh-TC/join?ref=36457687>**

如果页面显示英文，可以点击右上角的“地球”按钮，划到最后点击【简体中文】将页面切换成中文模式。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0a3364abc5_1709522690-14.jpg) 

2. 点击“通过邮箱/手机号码注册”按钮。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0646ef1454_1709522770-14.jpg) 

3. 输入邮箱号（可以使用国内163邮箱、qq邮箱或国外Outlook邮箱、Gmail邮箱等），或者是手机号码。如果是手机号码注册，需要将前面的区号更改为“**+86（中国）**”。再输入手机号码，勾选同意条款，点击【下一步】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/69928ca19e_1709523185-16.jpg) 

4. 根据要求进行真人认证。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/736cad05d9_1709524865-17.jpg) 

5. 接下来系统将向您的手机号码发送一个6位数字的验证码，去查找一下，填写好验证码，点击【提交】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4b48d37be7_1709619705-12.jpg) 

6. 设置密码。要求至少8个字符、至少1个数字和1个大写字母。设置完成后，点击“下一步”按钮。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6d83b0f023_1709525021-19.jpg) 

7. 如果是使用我的链接注册的，邀请码这里就自动显示出来。如果没有显示，可以手动填写邀请码**36457687**，使用此邀请码就能够在后续的交易中永久享受20%的手续费优惠。

注意后面一定要显示“对钩已绑定”才能够享受这个20%的优惠，点击【下一步】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d32277f383_1709525201-29.jpg) 

8. 账号注册成功。点击“是”保持登录状态。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5e8432d281_1709525303-30.jpg) 

# **第二步：币安实名认证**

账户注册完成之后，需要进行实名认证，不认证是不能使用的。

1. 进入账户之后，如果显示英文界面，可以点击右上角的“地球”按钮，往下翻，找到“简体中文”，把界面模式更改为中文。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6393fa0d17_1709525894-32.jpg) 

2. 点击“认证”按钮，进入身份认证页面。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6e7b6a24a2_1709526056-33.jpg) 

3. 居住信息选择“中国”，点击【继续】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0baa1239e9_1709526636-34.jpg) 

4. 国籍选择“中国”，填写姓名和出生年月日，点击【继续】。

这里要注意一定要填写真实的身份信息，因为后面还需要进行人脸识别。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/49ec5e4a9d_1709527322-35.jpg) 

5. 证件签发地区/国家选择“中国”，证件类型选择“身份证”，点击【继续】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9c93201db9_1709528334-36.jpg) 

6. 根据要求上传身份证件正反面照片，点击【继续】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e842004027_1709528546-37.jpg) 

7. 可以选择用摄像头拍照或上传照片，在电脑上完成整个身份认证过程，点击【继续】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/1aec72cf47_1709530402-38.jpg) 

8. 如果选择上传照片，那么上传的图片中细节一定要清晰可见，身份证件一定要拍得很完整，边边角角都要照顾到。最好是放在纯色背景下去拍，这样识别度会高一些。还要注意证件一定要是原件，并且在有效期内。点击【上传】分别对身份证件的正面（人像面）、反面（国徽面）进行上传，点击【继续】。再根据系统提示扫个脸就可以完成认证。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4893ba4dee_1709530881-39.jpg) 

# **第三步：账户安全设置**

虽然我们已经注册好账号，也进行实名认证了，但是我们现在这个账号还有安全隐患。为了进一步确保账户安全，我们还需要对账户进行安全设置。

1. 返回到币安首页，点击右上角的小人头像，找到【账户】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9cb7836426_1709532835-40.jpg) 

2. 点击【账户安全】，分别对身份验证器APP、邮箱和手机号进行设置。这三项都很重要，建议都要设置。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/677946b869_1709532993-41.jpg) 

3. 电话号码这一项，因为我们在注册的时候用的就是手机号，后面就会显示绿色对钩已绑定，我们不需要管。

邮箱验证这里，我们点击后面的【管理】，选择【添加邮箱】，输入验证码点击【提交】，根据提示要求输入邮箱号进行绑定。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6361a438e3_1709534003-44.jpg) 

4. 这里我们重点来说一下身份验证器的绑定。

点击身份验证器APP后面的【管理】，点击“启用身份验证器APP”，输入手机号验证码，点击【提交】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/833ae4ad3e_1709534532-45.jpg) 

5. 此时会出现验证器二维码和下方的16位密钥。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8004602669_1698308037-35.jpg) 

6. 打开谷歌验证器APP（谷歌验证器APP的下载方法，文章后面的部分会进行介绍），可以点击下图中的相机扫描上图二维码或输入上图二维码下方的16位密钥。两种方法均可，之后点击上图中的“下一步”按钮。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2c1ecaced3_1709534868-46.jpg) 

完成以上操作，您的谷歌验证器APP上便会出现6位数字验证码，将此验证码输入到身份验证器内，点击【提交】按钮，即可完成谷歌验证器的绑定。

谷歌验证器绑定成功以后，您下次登录账户或进行提现等操作的时候，除了要输入账户密码以外，还需要填入谷歌验证器验证码进行双重验证，确保您的账户资金安全。

# **第五步：购买USDT**

1. 点击币安首页左上角的“一键买币”，选择“自选区”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/11d42633af_1706151353-a.jpg) 

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d37ca7aa5f_1706151368-b.jpg) 

2. 点击“买入”，币种选择USDT，输入要购买的金额，后面的“CNY”就是人民币的意思，即我们要使用的法币。选择你习惯使用的支付方式。

此时系统会为我们筛选出来一些可交易的商家，商家的排价顺序都是由低到高的。因为我们想“买入”，当然想要买价格便宜，同时成交量多和好评率高的商家去购买。比如“佳智货商”这一家，它的成交量有一万四千多，好评率为100%，我们就可以跟这家进行购买，点击“购买USDT”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/fd64e64c27_1709537423-52.jpg) 

3. 再次输入一下要购买的金额，输入100，它会自动为我们算出来支付100人民币能够接收到13.45颗USDT。

注意一定要仔细阅读广告方的交易条款，根据要求给商家付款。确定没有问题后，点击【购买USDT】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7e94063315_1709537189-51.jpg) 

4. 订单生成。需要在规定的时间内，根据页面中商家所展示的付款方式给他付款，付款完成后，点击【我已付款，通知对方】，等待卖家放币给我们。

注意：

* 完成付款后再点击“我已付款，通知卖家”，没有付款就不要点；
* 不要在聊天框内与卖家发送任何关于买币的问题；
* 为了避免账户被冻结，一定要实名付款；
* 付款之前要核实收款人与订单上的是否一致。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5dc39d5c89_1709537795-53.jpg) 

5. 我们购买好的USDT就会到我们的资金账户里面，点击右上角“钱包”按钮，找到【资金账户】，这里可以查看余额。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b52ace7497_1709626070-13-1024x436.jpg) 

# **第六步：购买SOL**

1. 想要进行现货交易，购买SOL，需要把资金账户里的USDT划转到现货账户里面去。

找到“划转”，从“资金账户”到“现货账户”，币种选择“USDT”，全部划转，点击“确认”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2683ef57c0_1709626421-14-1024x560.jpg) 

2. 划转成功之后，点击上方的“交易”按钮，点击“现货交易”。现货交易就是币币交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7fd8f93f4a_1706154080-17.jpg) 

3. 选择我们交易的币种，我们想去购买SOL，就可以在上方搜索“SOL”。因为我们有USDT，所以我们点击USDT这一栏，选择“SOL/USDT”，意思就是我们使用USDT去兑换SOL。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9a85fce9fe_1710745401-18.jpg) 

4. 选择“市价”。市价就是他会以当前市场最合适的价格给我们。我们直接输入购买的金额，或者拉动滑杆选择购买的数量，点击【买入SOL】，就可以购买成功。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/67d9d80ffd_1710745575-19.jpg)
