# 欧易OKX交易所使用步骤

欧易算是中文区的热门交易所。个人觉得它的用户体验相对友好，比如充提币的到账速度就比其他家略微快一点。本篇文章将从注册、认证、交易买币三个流程，为大家全方位讲解欧易交易所的使用步骤。

# **第一步：创建账户**

用户注册在电脑端和手机端都可以操作。但由于接下来的账户认证只支持在手机端完成，我们就直接安排用手机来演示。

在手机浏览器中输入下方链接。这个链接带有邀请码，可以让家人们享受20%的手续费返佣。

**[https://www.okx.com](https://www.okx.com/join/76527935)**

进入官网页面。建议使用Gmail/Hotmail等海外邮箱进行注册。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/141c7c914a_1721108576-31.jpg) 

填写邮箱验证码。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/514dec3e72_1721108636-32.jpg) 

来到创建密码。这里要看清楚它的密码设置要求。每满足一个要求，前面就会打上绿色的小勾，点击“完成”，就完成了账户的创建。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9c2a45d45b_1721108769-33.jpg) 

# **第二步：账户KYC认证**

接下来是下载欧易APP进行身份认证。点击“本地下载”按钮，针对下载APP。苹果iOS的家人们需要切换美区或者香港ID进行下载。安卓的家人们直接点击下载安装包就可以了。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c60f98a83a_1721108968-34.jpg) 

装好之后打开APP。它会送一个盲盒，用刚才的账户登录，我这里领到了5U的返佣卡。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/2d6a2bdead_1721109308-38.jpg) 

接下来开始账户认证。点击左上角的头像，选择第一个“身份认证”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8150895a08_1721109438-41.jpg) 

先进行L1基础认证。点击“去认证”，国籍选择“中国”或者你实际的国籍。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3d2af48f9b_1721109639-46.jpg) 

输入准确的姓名和身份证号。只要信息是真实有效的，L1认证基本秒过。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e5bd724fc3_1721109736-48.jpg) 

我们再进入刚才的身份认证页。L2高级认证可做可不做。它与L1的差别在于额度的上限，是需要上传身份证照片的。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/183720568b_1721109947-54.jpg) 

# **第三步：买币交易**

认证完成以后，我们就可以进行买币交易了。回到欧易首页，点击“购买或充值数字货币”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/cbe781b51e_1721110030-55.jpg) 

点击“我想要买币”，选择“USDT”。我们要先用人民币去买USDT，再用USDT去买各种加密货币。这里的USDT相当于一个桥梁。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/7edefbf738_1721110200-58.jpg) 

我买100块等值的USDT，显示100块能买14.3个USDT。购买方式有三种，都可以。我选择的是“支付宝”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/b1d1786484_1721110352-61.jpg) 

这里需要绑定手机号。绑定完成之后，点击“继续交易”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d4e8295e6e_1721110770-64.jpg) 

点击【下一步】。点击“查看二维码”，卖家的支付宝会自动弹出来。接下来需要我们自己手动打开支付宝去扫码支付100块，就像我们去买东西，需要我们自己扫码付钱给老板。如果这边扫码失败，我们可以复制卖家的账号进行转账。需要我们自己去操作，平台不会跳转支付宝。转账完成以后，点击【我已转账，通知卖家】。这一步很重要，它就是告诉平台和卖家：我的钱已经转给你，快点收款把U给我。至此我们就完成了一次完整的买币交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3ae1f14a8d_1721111290-68.jpg) 

这边补充一句，第一次转账的小伙伴们可能会担心，私下转账会不会遇到卖家不承认收到钱的情况。这是不会的，大家放心。遇到这种问题直接联系欧易的客服。就比如说我自己第一次买币的时候，也不知道要点这个按钮，干等了一下午U都没有到账。后来联系客服，顺利帮我找回了这笔钱。所以大家一定要记得点那个按钮。我踩过的坑希望能够帮大家避开。
